# Assignment-4-Debugger
Debugger for ITP Assignment 4. Prints BoardData with its entries in an appropriate way

# Usage
1. Past the entire code from the `Debugger.java` into your assignment
2. Add method `logBoard` (written below) to the `Board` class (optional)
3. Call this method whenever you need; Debug your code
4. Do not forget to remove it before send the code to the codeforces (you may got plagiat warn)

Code for class `Board`:
```java
public void logBoard() {
    Debugger.logBoard(boardData, size);
}
```
